> Local Retrieval-Augmented Generation system — upload PDFs and ask contextual questions without paid APIs.

- 📂 Upload & index PDFs
- 🧮 Generate embeddings locally via Ollama

- 🏠 Fully local LLM inference — no paid APIs required

**Architecture:** `Client → FastAPI → FAISS → Ollama (Embeddings + LLM)`
